Apply this template to any rule (or paste JSON into the Templates-tab Sample pane) to inspect the parsed payload three different ways.

Tree — {% raw %}{{ __root | tree }}{% endraw %}

Collapsible tree. Click the triangles to expand/collapse. Best for large or deeply-nested payloads.

{{ __root | tree }}

Pretty — {% raw %}{{ __root | json: 2 }}{% endraw %}

Indented text. Best for medium payloads when you want to copy/paste fragments.

{{ __root | json: 2 }}

Compact — {% raw %}{{ __root | json }}{% endraw %}

One-line dump. Best for tiny payloads or quick "what's the exact shape?" checks.

{{ __root | json }}
